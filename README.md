Este projeto é uma versão personalizada do Window-Switcher, criado originalmente por Sigoden. Mantive a proposta e a base do projeto original, adicionando modificações próprias na interface, no comportamento e nas configurações para adaptar o aplicativo ao meu fluxo de trabalho no Windows. É, essencialmente, minha interpretação pessoal do projeto, construída sobre o excelente trabalho original e ajustada às minhas preferências de uso.
